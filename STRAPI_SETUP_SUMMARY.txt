🧪 HOW IT WORKS
═════════════════════════════════════════════════════════════════════════════

User visits page
    ↓
Application tries to fetch from Strapi
    ↓
    ├─ Success? → Use Strapi data
    └─ Failure? → Load fallback JSON
    ↓
Page renders with data
    ├─ Display all content
    ├─ Show images
    └─ Full functionality
